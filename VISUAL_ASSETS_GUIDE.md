## Creating Demo Graphics

### Using Canva (Free)
1. Go to canva.com
2. Use "GitHub Banner" template (1280x640)
3. Choose dark tech theme
4. Add shield/security icons
5. Use suggested color scheme
6. Export as PNG

### Using Figma (Free)
1. Create new design (1280x640)
2. Use gradient background (dark blue â†’ lighter blue)
3. Add icons from Feather Icons or Heroicons
4. Use fonts: Inter, Roboto, or Poppins
5. Export as PNG

### Using Terminal Screenshots
For CLI demos:
```bash
# Install carbon-cli for beautiful terminal screenshots
npm install -g carbon-now-cli

# Create screenshot
carbon-now sentinel analyze sample.exe --live
```

### Using Asciinema
Record terminal sessions:
```bash
# Install asciinema
pip install asciinema

# Record session
asciinema rec demo.cast

# Upload and get shareable link
asciinema upload demo.cast
```

## Example Workflows

### Capture Perfect CLI Screenshot

1. **Setup Terminal**
   ```bash
   # Use Windows Terminal or iTerm2 for best results
   # Set theme to One Dark or Dracula
   # Font: Fira Code or JetBrains Mono
   # Size: 14-16pt
   ```

2. **Run Command with Visual Output**
   ```bash
   sentinel analyze sample.exe --live
   ```

3. **Capture at Right Moment**
   - Wait for interesting events to populate
   - Ensure statistics show varied data
   - Capture full terminal window

4. **Edit Screenshot**
   - Crop excess whitespace
   - Add subtle shadow
   - Resize if needed
   - Optimize file size

### Create Animated Demo GIF

1. **Record Session**
   ```bash
   # Use asciinema or ttygif
   asciinema rec demo.cast
   
   # Convert to GIF
   asciicast2gif demo.cast demo.gif
   ```

2. **Optimize GIF**
   ```bash
   # Use gifsicle to reduce size
   gifsicle -O3 demo.gif -o demo-optimized.gif
   ```

3. **Add to README**
   ```markdown
   ![Demo](assets/demo.gif)
   ```
